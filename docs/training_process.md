# Training Process

This project is presented as a public case study and does not expose proprietary training data or production artifacts.

## Public workflow

The training process shown here is intended to demonstrate how the system would be developed.

### Step 1

Collect real estate listing descriptions that contain unit mix information.

### Step 2

Label unit counts, bedroom counts, and bathroom counts from the text.

### Step 3

Train the extraction model to identify relevant spans and relationships.

### Step 4

Evaluate extraction quality against held out examples.

### Step 5

Refine post processing rules to improve normalization and output consistency.

## Public repository note

This repository uses safe public examples and stub logic to demonstrate the pipeline structure without exposing sensitive data.
